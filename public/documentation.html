Movie API

This web application will provide its users with access to information about various movies, directors, and genres. Users will be able to sign up, update their personal information, and creat a list of their favorite movies.

Show All Movies

"/movies"

Method Type: GET

Description: This endpoint provides the user with the full list of movies listed by the API. It returns an array of objects (movies) and their parameters (details about each movie).

Request body data format: None

Response body data format: An array with a list of all movies. Each object contains data about one specific movie.

Example:

{
Title: "The Thing",
Description: "Based on the 1938 John W. Campbell Jr. novella Who Goes There?, it tells the story of a group of American researchers in Antarctica who encounter the eponymous "Thing", a parasitic extraterrestrial life-form that assimilates, then imitates, other organisms.",
Genre: { Name: "Horror" },
Director: { Name: "John Carpenter" },
ImageUrl: "https://media-cache.cinematerial.com/p/500x/i9cipbg7/the-thing-movie-poster.jpg?v=1456281372"
}

Example:

{
Name: "Science fiction",
Description: "Science fiction is a film genre that uses speculative, fictional science-based depictions of phenomena that are not fully accepted by mainstream science, such as extraterrestrial lifeforms, spacecraft, robots, cyborgs, interstellar travel or other technologies."

Example:

{

Birth/Death Year: "placeholder dates"

Adding Users

"/users"

Method Type: POST

Description: This endpoint allows the user to register. Returns a JSON object with sveral key-value pairs. "Name" returns a string, "favorite movies" returns an array, "id" returns a string.

Request body data format: JSON object containing data about the user.

Example:

{
Name: "James Mayfell",
favoriteMovies: [],
}

Response body data format: An object containing data about the user including a generated unique ID.

Example:

{
Name: "James Mayfell",
favoriteMovies: [],
id: "1"
}

Updating Username

"/users/[id]"

Method Type: POST

Description: This endpoint allows the user to update/change their username. Returns a JSON object with sveral key-value pairs. "Name" returns a string, "favorite movies" returns an array, "id" returns an integer.

Request body data format: JSON object containing data about the user.

Example:

{
Username: "newUserName"
}

Response body data format: An object containing data about the user including a generated unique ID.

Example:

{
Name: "James Mayfield",
favoriteMovies: [],
id: "1"
}

Add a Favorite Movie

"/users/[id]/movieTitle"

Method Type: POST

Description: This endpoint allows the user to add a movie to their favorites list. Returns a JSON object with user data including the updated array showing their favorite movies.

Request body data format: None

Response body data format: A message confirming the user has successfully updated their favorite movie list (array).

Example:

"movieTitle has been added to favorites list!"

Response Status Codes:

200: OK: movie added

400: Bad Request: movie not found

Remove a Favorite Movie

"/users/[id]/movieTitle"

Method Type: DELETE

Description: This endpoint allows the user to remove a movie from their favorites list. Returns a JSON object with user data including the updated array showing their favorite movies.

Request body data format: None

Response body data format: A message confirming the user has successfully removed a movie from their favorites list.

Example:

"movieTitle has been removed from favorites list."

Response Status Codes:

200: OK: Movie removed from array

400: Bad Request: Movie not on favorites list

Deleting a User

"/users/[id]"

Method Type: DELETE

Description: This endpoint allows the user to remove their account by way of ID number. Returns a message confirming that the account was deleted.

Request body data format: None

Response body data format: A message confirming the user has deleted their account.

Example:

"User {id number} has been deleted."

Response Status Codes:

200: OK: User has been deleted

400: Bad Request: User not found
