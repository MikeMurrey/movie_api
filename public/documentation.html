MyFlix API

This web application will provide its users with access to information about various movies, directors, and genres. Users will be able to sign up, update their personal information, and creat a list of their favorite movies.


Show All Movies

"/movies"

Method Type: GET

Description: This endpoint provides the user with the full list of movies listed by the API. It returns an array of objects (movies) and their parameters (details about each movie).

Query Parameters: None

Expected request data: Required: /movies

Example request:

localhost:8080/movies

Expected response data: An array with a list of all movies. Each object contains data about one specific movie.

Example response:

{

Adding Users

"/users"

Method Type: POST

Description: This endpoint allows the user to register. Returns a JSON object with sveral key-value pairs. "Name" returns a string, "favorite movies" returns an array, "id" returns a string.

Query Parameters: None

Expected request data: JSON object with data about the user (specifically their username).

Example request:

{
userName: "Darmok",
}

Expected response data: A JSON object containing data about the user including a generated unique ID.

Example response:

{
id: 1
userName: "Darmok",
favoriteMovies: [],
}

Response status codes:

201: User created

400: Bad Request- username required!


Updating Username

"/users/:id"

Method Type: POST

Description: This endpoint allows the user to update/change their username. Returns a JSON object with sveral key-value pairs. "Name" returns a string, "favorite movies" returns an array, "id" returns an integer.

Query Parameters: :id

Expected request data: JSON object with data about the user.

Example request:

{
userName: "newUserName"
}

Expected response data: A JSON object containing data about the user including a generated unique ID and their updated userName.

Example:

{
id: 1,
userName: "newUserName",
favoriteMovies: [],
}

Response status codes

200: userName updated

400: Bad Request- user does not exist


Add a Favorite Movie

"/users/:id/:movieTitle"

Method Type: POST

Description: This endpoint allows the user to add a movie to their favorites list. Returns a JSON object with user data including the updated array showing their favorite movies.

Query Parameters: :id, :movieTitle

Expected request data: Required: /id, /movieTitle

Example request:

localhost:8080/users/1/Alien

Expected response data: A message confirming the user has successfully updated their favorite movie list (array).

Example response:

"Alien has been added to favorites list!"

Response Status Codes:

200: OK: movie added

400: Bad Request: movie not found


Remove a Favorite Movie

"/users/:id/:movieTitle"

Method Type: DELETE

Description: This endpoint allows the user to remove a movie from their favorites list. Returns a JSON object with user data including the updated array showing their favorite movies.

Query Parameters: :id, :movieTitle

Expected request data: Required: /id, /movieTitle

Example request:

localhost:8080/users/1/Alien

Expected response data: A message confirming the user has successfully removed a movie from their favorites list.

Example response:

"Alien has been removed from favorites list."

Response Status Codes:

200: OK: Movie removed

400: Bad Request: Movie not on favorites list


Deleting a User

"/users/:id"

Method Type: DELETE

Description: This endpoint allows the user to remove their account by way of ID number. Returns a message confirming that the account was deleted.

Query Parameters: :id

Expected request data: Required: /id

Example request:

localhost:8080/users/1

Expected response data: A message confirming the user has deleted their account.

Example response:

"User {1} has been deleted."

Response Status Codes:

200: OK- User has been deleted

400: Bad Request- User not found
