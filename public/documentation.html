MyFlix API

This web application will provide its users with access to information about various movies, directors, and genres. Users will be able to sign up, update their personal information, and creat a list of their favorite movies.

Adding Users

"/users"

Method Type: POST

Description: This endpoint allows the user to register. Returns a JSON object with several key-value pairs. "Name" returns a string, "favorite movies" returns an array, "id" returns a string.

Query Parameters: None

Expected request data: JSON object with data about the user (Username, Password, and Email are required. Birthday and Favorite Movies are optional).

Example request:

{
"Username": "TestUser",
"Password": "TestPass",
"Email": "TestEmail@email.com"
}

Expected response data: A JSON object containing data about the user including a generated unique ID.

Example response:

{
"Username": "TestUser",
"Password": "TestPass",
"Email": "TestEmail@email.com",
"Birthday": null,
"_id": "633200c537a1f4befde940dd"
}

Response status codes:

201: User created (returns new user JSON object)

500: Error + error type


Updating User Info by Username

"/users/:Username"

Method Type: PUT

Description: This endpoint allows the user to update/change their information. Returns a JSON object with sveral key-value pairs. "Name" returns a string, "favorite movies" returns an array, "id" returns an integer.

Query Parameters: :Username

Expected request data: JSON object with data about the user.

Example request:

{
"Username": "FixName",
"Password": "FixPass",
"Email": "FixEmail@email.com",
"Birthday": "1999-08-18"
}

Expected response data: A JSON object containing data about the user including a generated unique ID and their updated userName.

Example:

{
"_id": "633200c537a1f4befde940dd",
"Username": "FixName",
"Password": "FixPass",
"Email": "FixEmail@email.com",
"Birthday": "1999-08-18T00:00:00.000Z",
"FavoriteMovies": []
}

Response status codes

200: Returns JSON object of user showing their updated information

500: Error + error type


Add a Favorite Movie

"/users/:Username/movies/:MovieID"

Method Type: POST

Description: This endpoint allows the user to add a movie to their favorites list. Returns a JSON object with user data including the updated array showing their favorite movies.

Query Parameters: :Username, :MovieID

Expected request data: Required: /Username, /MovieID

Example request:

localhost:8080/users/FixName/movies/632a449726de5951317fde2a

Expected response data: Updated JSON of user data showing movie added to favorites list

Example response:

{
"_id": "633200c537a1f4befde940dd",
"Username": "FixName",
"Password": "FixPass",
"Email": "FixEmail@email.com",
"Birthday": "1999-08-18T00:00:00.000Z",
"FavoriteMovies": ["632a448526de5951317fde2a"]
}

Response Status Codes:

200: Shows updated JSON

500: Error + error type


Remove a Favorite Movie

Expected response data: An updated JSON object showing the new contents of the FacoriteMovies array

Example response:

"Username": "TestUser",

"Email": "TestMail@email.com",

"FavoriteMovies": [ "632a449726de5951317fde2c" ]

Deleting a User by Username

"/users/:Username"

Method Type: DELETE

Description: This endpoint allows the user to remove their account. Returns a message confirming that the account was deleted.

Query Parameters: :Username

Expected request data: Required: /Username

Example request:

localhost:8080/users/FixName

Expected response data: A message confirming the user has deleted their account.

Example response:

"FixName was deleted"

Response Status Codes:

200: Username was deleted.

400: Username was not found.
